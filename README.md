# 📒 Smart Contact Manager (Python Project)

## 🚀 Project Description
A simple yet advanced Contact Management System built using Python. It allows users to add, view, edit, delete, and search contacts with a GUI interface and persistent storage.

---

## 💻 Features
- Add new contacts
- View saved contacts
- Edit contact details
- Delete contacts
- Search functionality
- GUI using Tkinter
- Data stored using JSON file

---

## 🛠 Tech Stack
- Python
- Tkinter (GUI)
- JSON (File Handling)
